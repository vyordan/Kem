voy a construir un compilador JIT y voy a usar llvm
mi compilador va tomar un archivo de texto y lo tiene que traducir a codigo IR que es un lenguaje intermedio, esto para luego usar LLVM para pasar al ejecutable en memoria
voy a tener sintaxis propia (totalmente en espaniol) y estas son las palabras reservadas que pienso incluir:

tipos de datos: entero, decimal, texto y booleano (funciones y arreglos unidimensionales de estos tipos de datos, no solo variables)
flujo de control: si, sino, mientras, hasta y paso (mientras va ser un bucle universal tipo como el de go, hasta y paso es para cuando se comprote como for (masomenos como el para en pseint))
bloques: inicio, procedimiento, estructura, funcion (procedimiento para cuando no devuelve y funcion para cuando si retorna) ----------DEBATIBLEEEE------------ mas palabras a aprender para estudiante -----------------
cgestion de datos: devolver, referencia y enlazar
logica: verdadero, falso, y, o, no
comenarios: comenario, comenario{lineas a coentar} || //, /**/ ---------DEBATIBLEEEE----------por que es intuitivo lo primero pero lo segundo es universal

estructura: {}, (), []
aritmetica +, -, *, /, %
asignacion =
acceso . (por el tema de las estructuras)
racionales ==, !=, <, >, >=, <=
los logicos son el y o y no que puse arriba

ahorita voy a instalar LLVM para generar codigo IR a mano para entender este lenguaje intermedio, nota de las dependencias que voy a instalar para tomar encuenta cuando haga mi readme xd

para arch linux necesito:

sudo pacman -S llvm clang lld
llvm: Las bibliotecas core y herramientas (incluye lli para ejecutar IR)
clang: El compilador C/C++ (útil para comparar) NO LO INSTALE
lld: El linker de LLVM
----------------------------------------------
sudo pacman -S cmake ninja   
para compilar proyectos con LLVM
-------------=-----------------------------------------
# Verificar versión
llvm-config --version

# Probar el intérprete de IR
echo 'define i32 @main() { ret i32 42 }' | lli
# Debería ejecutar silenciosamente (o mostrar 42 si agregas print)
----------------------------------------------------------
# Instalar cabeceras y bibliotecas de desarrollo
sudo pacman -S llvm-libs
